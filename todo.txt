URL  SHORTNER

Design a Url shortner services that takes in a valid URL and returns a shortned URL, 
redirecting the user to the priviously provided URL.

Also, keep track of total visits/clicks on the url.

Routes:-

POST /URL - Generate a new short URL and Returns the shortened URL 
            in the format example.com/random-id

GET /:id - Redirects the user to the original URL.

GET /URL/analyticals/:id - Return the clicks for the provided short id.
